      Fonction polynôme

      Définition

      Une fonction polynôme de degré 2 est une fonction f f f définie sur R\mathbb{R}R dont l’expression algébrique peut être mise sous la forme

      f(x)=ax2+bx+c, f(x) = ax^2 + bx + c, f(x)=ax2+bx+c,


      où a,b,c a, b, c a,b,c sont des réels avec a≠0 a \neq 0 a=0.
      Les réels a,b,c a, b, c a,b,c sont appelés coefficients de la fonction polynôme.

      Remarque

      L’expression ax2+bx+c ax^2 + bx + c ax2+bx+c est dite forme développée de f(x) f(x) f(x) ou trinôme du second degré.

      Forme canonique

      Propriétés et définition

      Toute fonction polynôme f f f de degré 2 de forme développée ax2+bx+c ax^2 + bx + c ax2+bx+c, admet une écriture de la forme

      f(x)=a(x−α)2+β, f(x) = a(x-\alpha)^2 + \beta, f(x)=a(x−α)2+β,


      où α=−b2a\alpha = \frac{-b}{2a}α=2a−b​ et β=f(α)\beta = f(\alpha)β=f(α).
      Cette écriture est la forme canonique de la fonction polynôme.

      Variation et représentation graphique

      Soit f f f une fonction polynôme de degré 2 telle que, pour tout x x x,

      f(x)=ax2+bx+c,a≠0. f(x) = ax^2 + bx + c, \quad a \neq 0. f(x)=ax2+bx+c,a=0.


      On pose α=−b2a\alpha = -\frac{b}{2a}α=−2ab​ et β=f(α)\beta = f(\alpha)β=f(α).

      Théorème

      • Si a>0 a > 0 a>0, alors f f f est strictement décroissante sur ]−∞;α] ]-\infty ; \alpha] ]−∞;α] et strictement croissante sur [α;+∞[ [\alpha ; +\infty[ [α;+∞[.
        f f f admet un minimum égal à β\betaβ atteint en x=α x = \alpha x=α.
      • Si a<0 a < 0 a<0, alors f f f est strictement croissante sur ]−∞;α] ]-\infty ; \alpha] ]−∞;α] et strictement décroissante sur [α;+∞[ [\alpha ; +\infty[ [α;+∞[.
        f f f admet un maximum égal à β\betaβ atteint en x=α x = \alpha x=α.

      Propriété (admise)

      Dans un repère du plan, la courbe représentative de f f f est une parabole de sommet S(α;β) S(\alpha ; \beta) S(α;β) qui admet pour axe de symétrie la droite d’équation x=α x = \alpha x=α.

      Exercice résolu : Déterminer la forme canonique

      Exercice résolu

      Déterminer la forme canonique du trinôme f(x)=−x2+2x−5 f(x) = -x^2 + 2x - 5 f(x)=−x2+2x−5.

      Solution commentée

      Première méthode
      f f f est une fonction polynôme de degré 2, de coefficients a=−1,b=2,c=−5 a = -1, b = 2, c = -5 a=−1,b=2,c=−5.
      Son écriture canonique est −(x−α)2+β-(x-\alpha)^2 + \beta−(x−α)2+β, avec :

      α=−b2a=−2−2=1,β=f(1)=−4. \alpha = \frac{-b}{2a} = \frac{-2}{-2} = 1, \quad \beta = f(1) = -4. α=2a−b​=−2−2​=1,β=f(1)=−4.


      Donc pour tout réel x x x,

      f(x)=−(x−1)2−4. f(x) = -(x-1)^2 - 4. f(x)=−(x−1)2−4.

      Deuxième méthode
      On écrit f(x)=−(x2−2x+5) f(x) = -(x^2 - 2x + 5) f(x)=−(x2−2x+5) en factorisant par a=−1 a = -1 a=−1.
      Puis, en remarquant que x2−2x=(x−1)2−1 x^2 - 2x = (x-1)^2 - 1 x2−2x=(x−1)2−1, on obtient :
      f(x)=−[(x−1)2−1+5]=−(x−1)2−4. f(x) = -\big[(x-1)^2 - 1 + 5\big] = -(x-1)^2 - 4. f(x)=−[(x−1)2−1+5]=−(x−1)2−4.

      Exercice résolu : Identifier la forme d’une fonction polynôme de degré 2

      Exercice résolu

      Sans calculatrice, associer chaque fonction polynôme ci-dessous à la parabole qui la représente :

      f(x)=−1+3(x−3)2,g(x)=−1−0,25(x−3)2, f(x) = -1 + 3(x-3)^2, \quad g(x) = -1 - 0,25(x-3)^2, f(x)=−1+3(x−3)2,g(x)=−1−0,25(x−3)2,
