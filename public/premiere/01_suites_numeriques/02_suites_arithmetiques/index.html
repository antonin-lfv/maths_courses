      Suites arithmétiques

      Définition

      Définition

      Soit u0u_0u0​ un nombre réel.
      Une suite (un)(u_n)(un​) de premier terme u0u_0u0​ est arithmétique s’il existe un nombre réel rrr tel que, pour tout entier naturel nnn,

      un+1=un+r. u_{n+1} = u_n + r. un+1​=un​+r.


      Le nombre rrr est appelé raison de la suite (un)(u_n)(un​).

      Remarque

      Une suite (un)(u_n)(un​) est arithmétique si, pour passer d’un terme au suivant, on ajoute toujours le même nombre ou encore si la différence un+1−unu_{n+1} - u_nun+1​−un​ ne dépend pas de nnn.

      Exemple

      Soit la suite (un)(u_n)(un​) définie par u0=3u_0 = 3u0​=3 et, pour tout entier naturel nnn,

      un+1=un+5 u_{n+1} = u_n + 5 un+1​=un​+5


      On passe d’un terme au suivant en ajoutant 5. Ainsi u1=8u_1 = 8u1​=8, u2=13u_2 = 13u2​=13, u3=18…u_3 = 18 \ldotsu3​=18…
      La suite (un)(u_n)(un​) est arithmétique de premier terme 3 et de raison 5.

      Propriété

      (un)(u_n)(un​) est la suite arithmétique de premier terme u0u_0u0​ et de raison rrr si et seulement si, pour tout entier naturel nnn,

      un=u0+nr. u_n = u_0 + nr. un​=u0​+nr.

      Remarques

      • La propriété précédente peut être utilisée avec d’autres termes que u0u_0u0​ :
        un=u1+(n−1)r=u2+(n−2)r=… u_n = u_1 + (n-1)r = u_2 + (n-2)r = \ldots un​=u1​+(n−1)r=u2​+(n−2)r=…
        et, de façon générale pour ppp entier naturel,
        un=up+(n−p)r. u_n = u_p + (n-p)r. un​=up​+(n−p)r.
      • De la relation de récurrence un+1=un+ru_{n+1} = u_n + run+1​=un​+r, on peut passer à la formule explicite un=u0+nru_n = u_0 + nrun​=u0​+nr.
      • Pour une suite arithmétique, on a un=f(n)u_n = f(n)un​=f(n), où fff est la fonction affine f(x)=u0+xrf(x) = u_0 + xrf(x)=u0​+xr.
        Dans un repère, les points de coordonnées (n;un)(n ; u_n)(n;un​) sont alignés.
      Exemple

      Soit la suite (un)(u_n)(un​) définie par u0=−1u_0 = -1u0​=−1 et, pour tout entier naturel nnn :

      un+1=un+1,5 u_{n+1} = u_n + 1,5 un+1​=un​+1,5


      La suite (un)(u_n)(un​) est arithmétique de premier terme u0=−1u_0 = -1u0​=−1 et de raison 1,5.
      Donc, pour tout entier naturel nnn, on a :

      un=u0+nr=−1+1,5n u_n = u_0 + nr = -1 + 1,5n un​=u0​+nr=−1+1,5n

      Somme des premiers entiers

      Propriété

      Pour tout entier naturel nnn non nul, on a :

      1+2+3+…+n=n(n+1)2. 1 + 2 + 3 + \ldots + n = \frac{n(n+1)}{2}. 1+2+3+…+n=2n(n+1)​.

      Remarque

      Il s’agit de la somme des nnn premiers termes de la suite arithmétique (un)(u_n)(un​) de premier terme u0=1u_0 = 1u0​=1 et de raison 1.

      Reconnaître une suite arithmétique

      Exercice résolu 1 — Reconnaître une suite arithmétique

      1. La suite (un)(u_n)(un​), définie par un=n2u_n = n^2un​=n2 pour tout entier naturel nnn, est-elle arithmétique ?
      2. La suite (vn)(v_n)(vn​), définie par vn=n2−(n+1)2v_n = n^2 - (n+1)^2vn​=n2−(n+1)2 pour tout entier naturel nnn, est-elle arithmétique ?
      Solution commentée

      1. On commence par calculer les premiers termes de la suite (un)(u_n)(un​) :

      u0=0 ; u1=1 ; u2=4 u_0 = 0 \ ; \ u_1 = 1 \ ; \ u_2 = 4 u0​=0 ; u1​=1 ; u2​=4


      On a u1−u0=1u_1 - u_0 = 1u1​−u0​=1 et u2−u1=3u_2 - u_1 = 3u2​−u1​=3, donc u1−u0≠u2−u1u_1 - u_0 \neq u_2 - u_1u1​−u0​=u2​−u1​.
      La suite (un)(u_n)(un​) n’est donc pas arithmétique.

      2. On commence par calculer les premiers termes de la suite (vn)(v_n)(vn​) :

      v0=02−(0+1)2=−1 ; v1=12−(1+1)2=−3 ; v2=22−(2+1)2=−5 v_0 = 0^2 - (0+1)^2 = -1 \ ; \ v_1 = 1^2 - (1+1)^2 = -3 \ ; \ v_2 = 2^2 - (2+1)^2 = -5 v0​=02−(0+1)2=−1 ; v1​=12−(1+1)2=−3 ; v2​=22−(2+1)2=−5


      Il semble que la suite (vn)(v_n)(vn​) soit arithmétique de raison −2-2−2.

      Pour le prouver, on montre que la différence entre deux termes consécutifs quelconques est égale à −2-2−2.
      Pour tout entier naturel nnn, on a :

      vn+1−vn=[(n+1)2−(n+2)2]−[n2−(n+1)2] v_{n+1} - v_n = [(n+1)^2 - (n+2)^2] - [n^2 - (n+1)^2] vn+1​−vn​=[(n+1)2−(n+2)2]−[n2−(n+1)2]


      =(n2+2n+1−n2−4n−4)−(n2−n2−2n−1) = (n^2 + 2n + 1 - n^2 - 4n - 4) - (n^2 - n^2 - 2n - 1) =(n2+2n+1−n2−4n−4)−(n2−n2−2n−1)


      =(−2n−3)−(−2n−1)=−2n−3+2n+1=−2 = (-2n - 3) - (-2n - 1) = -2n - 3 + 2n + 1 = -2 =(−2n−3)−(−2n−1)=−2n−3+2n+1=−2

      Ainsi, pour tout entier naturel nnn, vn+1−vn=−2v_{n+1} - v_n = -2vn+1​−vn​=−2.
      Donc la suite (vn)(v_n)(vn​) est arithmétique de raison -2.

      Remarque : on peut également prouver que (vn)(v_n)(vn​) est arithmétique en montrant que

      vn=n2−(n+1)2=−2n−1=f(n) v_n = n^2 - (n+1)^2 = -2n - 1 = f(n) vn​=n2−(n+1)2=−2n−1=f(n)


      avec fff affine.

      Étudier une suite arithmétique

      Exercice résolu 2 — Étudier une suite arithmétique

      Soit la suite (un)(u_n)(un​) définie par u0=−7u_0 = -7u0​=−7 et, pour tout entier naturel nnn,

      un+1=un+4. u_{n+1} = u_n + 4. un+1​=un​+4.
      1. Donner la formule explicite de unu_nun​. En déduire la valeur de u21u_{21}u21​.
      2. Un terme de la suite vaut-il 2019 ?
      Solution commentée

      1. La suite (un)(u_n)(un​) est arithmétique de premier terme u0=−7u_0 = -7u0​=−7 et de raison 4.
      Donc, pour tout entier naturel nnn, on a :

      un=u0+nr=−7+4n. u_n = u_0 + nr = -7 + 4n. un​=u0​+nr=−7+4n.


      En particulier :

      u21=−7+4×21=77. u_{21} = -7 + 4 \times 21 = 77. u21​=−7+4×21=77.

      2. On cherche s’il existe un entier naturel nnn tel que un=2019u_n = 2019un​=2019.
      On résout l’équation :

      un=2019  ⟺  −7+4n=2019  ⟺  4n=2026  ⟺  n=20264=506,5. u_n = 2019 \iff -7 + 4n = 2019 \iff 4n = 2026 \iff n = \frac{2026}{4} = 506,5. un​=2019⟺−7+4n=2019⟺4n=2026⟺n=42026​=506,5.

      Or nnn doit être un entier naturel, donc aucun terme de la suite n’est égal à 2019.
