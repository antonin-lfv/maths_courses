      Définition d’une suite numérique

      Définition

      Une suite numérique est une fonction u:n↦u(n)u : n \mapsto u(n)u:n↦u(n) définie sur N\mathbb{N}N (ou seulement pour n≥kn \geq kn≥k avec kkk entier naturel) et à valeurs dans R\mathbb{R}R.
      Le nombre réel u(n)u(n)u(n), noté unu_nun​ (se lit « uuu indice nnn »), est appelé le terme de rang nnn ou le terme général de la suite. On note cette suite (un)(u_n)(un​).

      Une suite (un)(u_n)(un​) peut être représentée graphiquement par le nuage de points de coordonnées (n;un)(n;u_n)(n;un​).

      Exemple
      La liste 50;25;12,5;6,25…50 ; 25 ; 12,5 ; 6,25 \ldots50;25;12,5;6,25… définit les premiers termes de la suite (un)(u_n)(un​) telle que u0=50u_0 = 50u0​=50, u1=25u_1 = 25u1​=25, u2=12,5u_2 = 12,5u2​=12,5, u3=6,25…u_3 = 6,25 \ldotsu3​=6,25….
      On dit que 50 est le terme de rang 0 ; 25 est le terme de rang 1 ; 12,5 est le terme de rang 2, etc.

      Suite définie par une formule explicite un=f(n)u_n = f(n)un​=f(n)

      Définition

      Une suite est définie par une formule explicite lorsque unu_nun​ s’exprime en fonction de l’entier nnn.
      Dans ce cas, on peut calculer chaque terme unu_nun​ directement à partir de son rang nnn.

      Exemples

      • Pour tout entier naturel nnn, on donne un=2nu_n = 2nun​=2n.
        u0=2×0=0 ; u1=2×1=2u_0 = 2 \times 0 = 0 \ ; \ u_1 = 2 \times 1 = 2u0​=2×0=0 ; u1​=2×1=2.
        u2=2×2=4 ;…; u20=2×20=40u_2 = 2 \times 2 = 4 \ ; \ldots ; \ u_{20} = 2 \times 20 = 40u2​=2×2=4 ;…; u20​=2×20=40.

      • Pour tout entier naturel n≥1n \geq 1n≥1, on donne vn=n−1v_n = \sqrt{n - 1}vn​=n−1​.
        v1=1−1=0v_1 = \sqrt{1 - 1} = 0v1​=1−1​=0 (le premier terme ici est v1v_1v1​ et non v0v_0v0​) ;
        v2=2−1=1 ;…; v17=17−1=4v_2 = \sqrt{2 - 1} = 1 \ ; \ldots ; \ v_{17} = \sqrt{17 - 1} = 4v2​=2−1​=1 ;…; v17​=17−1​=4.

      Suite définie par une relation de récurrence un+1=f(un)u_{n+1} = f(u_n)un+1​=f(un​)

      Définition

      Une suite est définie par une relation de récurrence lorsqu’elle est définie par la donnée de :

      • son premier terme ;
      • une relation qui permet de calculer chaque terme à partir du précédent.

      Dans ce cas, pour calculer chaque terme unu_nun​, il faut avoir calculé tous les termes qui le précèdent.

      Exemples

      • On définit la suite (un)(u_n)(un​) par u0=5u_0 = 5u0​=5 et chaque terme est le triple de son précédent.
        u0=5 ; u1=3u0=3×5=15 ; u2=3u1=3×15=45…u_0 = 5 \ ; \ u_1 = 3u_0 = 3 \times 5 = 15\ ;\ u_2 = 3u_1 = 3 \times 15 = 45 \ldotsu0​=5 ; u1​=3u0​=3×5=15 ; u2​=3u1​=3×15=45…

      • On définit la suite (vn)(v_n)(vn​) par v0=3v_0 = 3v0​=3 et, pour tout entier naturel nnn,
        vn+1=4vn−6v_{n+1} = 4v_n - 6vn+1​=4vn​−6.
        v0=3 ; v1=4v0−6=4×3−6=6 ; v2=4v1−6=4×6−6=18…v_0 = 3 \ ; \ v_1 = 4v_0 - 6 = 4 \times 3 - 6 = 6 \ ; \ v_2 = 4v_1 - 6 = 4 \times 6 - 6 = 18 \ldotsv0​=3 ; v1​=4v0​−6=4×3−6=6 ; v2​=4v1​−6=4×6−6=18…

      Remarque

      Il existe d’autres modes de génération d’une suite comme par exemple un algorithme ou encore un dénombrement lié à une suite de motifs géométriques.

      Calculer des termes d’une suite

      Exercice résolu 1 — Calculer des termes d’une suite

      1. Soit la suite (un)(u_n)(un​) définie pour tout entier naturel nnn par :

        >un=3n2−2n+1> > u_n = 3n^2 - 2n + 1 > >un​=3n2−2n+1>


        a. Calculer le terme de rang 5, puis le 10e terme.
        b. Déterminer l’expression en fonction de nnn des termes un+1u_{n+1}un+1​ et u2nu_{2n}u2n​.

      2. Soit la suite (vn)(v_n)(vn​) définie par v0=1v_0 = 1v0​=1 et, pour tout entier naturel nnn,

        >vn+1=0,5vn+4.> > v_{n+1} = 0,5v_n + 4. > >vn+1​=0,5vn​+4.>


        Calculer v3v_3v3​.

      3. Soit la suite (wn)(w_n)(wn​) définie par w0=32w_0 = \tfrac{3}{2}w0​=23​ et, pour tout entier naturel nnn,

        >wn+1=2wn(1−wn)+2.> > w_{n+1} = 2w_n(1 - w_n) + 2. > >wn+1​=2wn​(1−wn​)+2.>


        Calculer w2w_2w2​.

      Solution commentée

      1.
      a. Le terme de rang 5 est u5u_5u5​. On remplace nnn par 5 :

      u5=3×52−2×5+1=66 u_5 = 3 \times 5^2 - 2 \times 5 + 1 = 66 u5​=3×52−2×5+1=66


      Le premier terme étant de rang 0, le 10e terme est de rang 9 :

      u9=3×92−2×9+1=226 u_9 = 3 \times 9^2 - 2 \times 9 + 1 = 226 u9​=3×92−2×9+1=226

      b.

      un+1=3(n+1)2−2(n+1)+1=3n2+6n+3−2n−2+1=3n2+4n+2 u_{n+1} = 3(n+1)^2 - 2(n+1) + 1 = 3n^2 + 6n + 3 - 2n - 2 + 1 = 3n^2 + 4n + 2 un+1​=3(n+1)2−2(n+1)+1=3n2+6n+3−2n−2+1=3n2+4n+2


      u2n=3(2n)2−2(2n)+1=12n2−4n+1 u_{2n} = 3(2n)^2 - 2(2n) + 1 = 12n^2 - 4n + 1 u2n​=3(2n)2−2(2n)+1=12n2−4n+1

      2. Pour obtenir v3v_3v3​, il faut calculer tous les termes qui le précèdent :

      v1=v0×0,5+4=0,5×1+4=4,5 v_1 = v_0 \times 0,5 + 4 = 0,5 \times 1 + 4 = 4,5 v1​=v0​×0,5+4=0,5×1+4=4,5


      v2=v1×0,5+4=0,5×4,5+4=6,25 v_2 = v_1 \times 0,5 + 4 = 0,5 \times 4,5 + 4 = 6,25 v2​=v1​×0,5+4=0,5×4,5+4=6,25


      v3=v2×0,5+4=0,5×6,25+4=7,125 v_3 = v_2 \times 0,5 + 4 = 0,5 \times 6,25 + 4 = 7,125 v3​=v2​×0,5+4=0,5×6,25+4=7,125

      3. Pour obtenir w2w_2w2​, il faut calculer tous les termes qui le précèdent :

      w1=2w0(1−w0)+2=2×32(1−32)+2=3×(−12)+2=−32+2=12 w_1 = 2w_0(1 - w_0) + 2 = 2 \times \tfrac{3}{2}(1 - \tfrac{3}{2}) + 2 = 3 \times (-\tfrac{1}{2}) + 2 = -\tfrac{3}{2} + 2 = \tfrac{1}{2} w1​=2w0​(1−w0​)+2=2×23​(1−23​)+2=3×(−21​)+2=−23​+2=21​


      w2=2w1(1−w1)+2=2×12(1−12)+2=1×12+2=52 w_2 = 2w_1(1 - w_1) + 2 = 2 \times \tfrac{1}{2}(1 - \tfrac{1}{2}) + 2 = 1 \times \tfrac{1}{2} + 2 = \tfrac{5}{2} w2​=2w1​(1−w1​)+2=2×21​(1−21​)+2=1×21​+2=25​
